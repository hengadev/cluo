## Prompt — Team Page (With Category Filtering)

> **Design a “Team” page that presents the practitioners/coaches behind the services.**
> This page is part of an existing marketing website and should visually align with the already-designed navigation, hero, and services pages.
>
> ---
>
> ### Page intent
>
> The goal of this page is to:
>
> * Build trust
> * Humanize the company
> * Help users choose the right professional for their needs
>
> The tone should feel:
>
> * Calm
> * Professional
> * Human and caring
> * Healthcare / wellness oriented
>
> ---
>
> ### Page structure
>
> #### 1. Hero section (text-only)
>
> * Large headline introducing the team (e.g. “Une équipe engagée pour votre bien-être”)
> * Short supporting paragraph explaining:
>
>   * The team’s philosophy
>   * Their complementary expertise
> * No imagery in the hero
> * Clean, airy layout with strong typography
>
> ---
>
> #### 2. Two-column layout (desktop)
>
> ##### Left column — Category filters
>
> A vertical list of categories used to filter team members:
>
> * Tous les profils
> * Bodywork & massage
> * Coaching mental
> * Entraînement physique
>
> Requirements:
>
> * Clear selected state
> * Simple, non-distracting styling
> * Feels consistent with the Services page filtering
>
> ---
>
> ##### Right column — Team members listing
>
> Team members are grouped visually by category when “Tous les profils” is selected.
>
> Each category section contains:
>
> * Category title
> * Very short descriptive sentence (optional, subtle)
>
> Under each category:
>
> * A list/grid of **team member cards**
>
> ---
>
> ### Team member card design
>
> Each card should include:
>
> * Profile photo (important)
> * Full name
> * Primary role / specialty
> * 1–2 key expertise tags
>
> Optional but encouraged:
>
> * A short quote or value statement (very concise)
>
> Primary interactions:
>
> * “Voir le profil” (primary CTA)
> * Optional secondary CTA: “Réserver une séance”
>
> The card should feel:
>
> * Approachable
> * Professional
> * Human
>
> Avoid anything that feels like:
>
> * A marketplace listing
> * A social network profile
>
> ---
>
> ### Filtering behavior
>
> * Selecting a category:
>
>   * Filters visible team members
>   * Preserves page structure
> * The interaction should feel:
>
>   * Instant
>   * Calm
>   * Non-technical
>
> ---
>
> ### Responsive behavior (mobile-first considerations)
>
> On mobile:
>
> * The category filter becomes:
>
>   * A horizontal scrollable selector, or
>   * A collapsible filter control
> * Team member cards stack vertically
> * Profile photos remain clearly visible
>
> The mobile experience should emphasize:
>
> * Human connection
> * Readability
> * Easy access to profiles
>
> ---
>
> ### Visual & UX constraints
>
> * Match the existing visual language of the site
> * Neutral, calming color palette
> * Generous spacing
> * Strong typographic hierarchy
> * No heavy borders or aggressive contrasts
>
> The page should feel trustworthy, not promotional.
>
> ---
>
> Design this page as a natural continuation of the Services page, reusing interaction patterns while adapting them to a more human-centric purpose.
